# PasteboardPro Plugin Roadmap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver independent ATools and ZTools PasteboardPro plugins with Paste 6-style macOS interaction, shared behavior contracts, and encrypted cross-host WebDAV synchronization.

**Architecture:** A self-contained plugin workspace in `ZTools-plugins/plugins/pasteboard-pro` owns the framework-agnostic TypeScript core, Svelte ATools UI, Vue ZTools UI, fixtures, and ZTools native helper. ATools adds a Rust pasteboard runtime, native shelf window, secure storage, TaskRun/MCP tools, and a narrow plugin bridge. The four phase plans below are executed in dependency order.

**Tech Stack:** TypeScript, pnpm workspaces, Vitest, Svelte 5, Vue 3, Vite, Rust, SQLite, Tauri 2, macOS AppKit/Vision/Accessibility, Node/Electron, AES-256-GCM, scrypt, WebDAV, Playwright.
