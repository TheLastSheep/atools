# 剪贴板本地检索

## 何时使用

当用户要查找刚复制过的链接、错误日志、token 片段、命令或临时文本，并强调不要上传云端时使用。

## 推荐工具顺序

1. `search_clipboard`：在本地剪贴板文本里搜索关键词。
2. 如需要定位关联文件，再使用 `find_local_files` 在用户确认的目录内搜索。
3. 如需要回填结果，由 Agent 给出文本摘要，涉及写剪贴板时必须单独确认。

## 权限

需要 `clipboard_read`。写回剪贴板时需要 `clipboard_write`，首版不要自动写回。

## 注意事项

- 首版 `search_clipboard` 只搜索当前剪贴板文本，不等同于完整剪贴板历史。
- 如果内容看起来像 API key、密码或 cookie，只返回脱敏摘要，除非用户明确要求显示完整内容。
- 不要把剪贴板内容发送给远程模型或远程服务。
